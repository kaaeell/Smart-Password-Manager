# 🔐 Smart Password Manager (CLI)

A simple command-line password manager built with Python.
I made this project to practice file handling and build something actually useful.

## 🚀 Features
- Master password login
- Add new passwords
- View saved passwords
- Search passwords
- Delete passwords

## 🧰 Technologies
- Python
- JSON (for storage)
- getpass (for hidden input)

## ▶️ How to Run
1. Make sure Python is installed   #btw name it main so it will be easy to run :)
2. Run the program:

   python main.py

## 📁 Data Storage
Passwords are stored in a file called `data.json`.

## ⚠️ Note
This is version 1, so passwords are NOT encrypted yet.
In future versions, I will add encryption for better security.
